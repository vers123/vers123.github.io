## 项目概述 / Project Overview

**Vers123 Game Navigation Website** - A clean and beautiful game navigation website providing official navigation links for miHoYo games and other popular games.

This website is designed to help players quickly access official resources for various games, including official websites, download links, community forums, and more.

---

## 项目特点 / Project Features

- **Clean and Beautiful Interface**: Modern UI design with smooth animations
- **Responsive Design**: Perfect display on various devices (PC, tablet, mobile)
- **Fast Loading**: Optimized code structure for quick page loading
- **Easy Navigation**: Clear categorization for quick access to required resources
- **Multi-game Support**: Currently supports miHoYo games and other popular games
